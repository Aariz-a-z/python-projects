# Password Generator 

## Description
A simple Python program that generates strong and random passwords based on user-defined length.

## Features
- User-defined password length
- Uses letters, numbers, and symbols
- Generates strong passwords

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the program:
   python password_generator.py

## Sample Output
Enter password length: 12  
Generated Password: A@9f#K2!xP$Q
