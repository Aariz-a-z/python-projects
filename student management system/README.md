## Student Management System

## Description
A Python-based system to manage student records such as name, roll number, and course using file storage.

## Features
- Add new student records
- View all student records
- Data stored persistently in a file

## Technologies Used
- Python 3
- File handling
- Object-Oriented Programming (OOP)

##  How to Run
1. Open terminal in the project folder
2. Run:
   python student.py

## Sample Output
1. Add Student  
2. View Students  
3. Exit
