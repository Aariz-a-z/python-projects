# Authentication System

## Description
A simple authentication system that allows users to sign up and log in using hashed passwords.

## Features
- User signup
- User login
- Password hashing for security

## Technologies Used
- Python 3
- hashlib module
- File handling

##  How to Run
1. Open terminal in project folder
2. Run:
   python auth.py

## Security
- Passwords are stored using SHA-256 hashing
